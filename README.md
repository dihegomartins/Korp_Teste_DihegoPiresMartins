# Desafio Técnico - Gestão de Notas Fiscais e Estoque 🚀

Repositório destinado ao desenvolvimento do desafio técnico para a **Korp**. O projeto consiste em uma solução de microsserviços para gerenciar o faturamento de notas fiscais e o controle de estoque de produtos.

## 🛠️ Tecnologias Utilizadas

* **Backend:** Go (Golang) - Focado em performance e concorrência.
* **Frontend:** Angular - Arquitetura SPA organizada sob o padrão MVC, garantindo alta performance e manutenibilidade..
* **Banco de Dados:** SQL Server - Persistência de dados relacional.
* **Arquitetura:** Microsserviços com comunicação via API REST.

## 🏗️ Arquitetura do Banco de Dados
A modelagem foi estruturada para garantir a integridade entre notas e itens, com regras de status e controle de saldo.
- [x] Criação de tabelas (Produtos, NotasFiscais, Itens).
- [x] Testes de integridade e relacionamentos via SQL.
